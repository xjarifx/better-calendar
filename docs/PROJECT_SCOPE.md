# Project Scope

## MVP Scope

### Included
- AI-powered event extraction from unstructured text
- Calendar view to display events
- Confirmation/edit step before saving
- Multiple notices at once
- Multi-user support (each user sees own events only)

### Out of Scope
- Google Calendar sync
- Mobile app
- Event reminders/notifications
- Event sharing with others
- Recurring events
- Text-based stack view (removed)
- Feedback/bug report feature (postponed)
