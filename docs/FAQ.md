# FAQ

### What does this software do?
A better alternative to Google Calendar. In Google Calendar, users must add events manually, which is boring, time-consuming, and error-prone. This app introduces a new section where users can simply paste event notices they receive, and AI will automatically extract the details and add them to the calendar.

### Can it extract multiple notices at once?
Yes.

### What AI will be used?
For the MVP, we will use OpenRouter's API to access freely available AI models.

### Will there be a system prompt?
Yes, a system prompt will instruct the AI to extract events from the pasted text. The system prompt will be designed alongside real data testing.

### Does the pasted text need to follow a specific format?
Not at all — that's the beauty of it. Users can paste text in any format, and the system prompt will be able to extract the relevant information.

### How is accuracy handled?
Users will see a confirmation/edit step before events are saved, since AI can occasionally misread dates, times, or other details. This ensures accuracy while keeping the experience fast.

### Can multiple people use this app?
Yes, each user has their own account (username + password) and can only see their own events. Friends can create accounts and use the app independently.
